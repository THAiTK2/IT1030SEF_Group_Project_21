# IT1030SEF_Group_Project_21

# Health and fitness tracking app
A mobile-first web app for daily health tracking.

---

## Core Features
- Onboarding and login flow
- Home tracking (water, activity, calories)
- Workouts filters with inline video preview
- Metrics (BMI, calorie target, chart + summary)
- Profile edit and persistence
- Manual light/dark theme switch

---
